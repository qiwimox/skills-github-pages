# Лабораторная №6
## GitHub Pages
### Шаг 2. Настройка главной страницы
1. Найдите index.mdфайл в этой mainветке.
2. В правом верхнем углу откройте редактор файлов.
3. Введите контент, который вы хотите разместить на главной странице. На этой странице можно использовать форматирование Markdown.
4. (необязательно) Вы также можете изменить title:его или оставить как есть.
5. Зафиксируйте изменения в mainветке.
